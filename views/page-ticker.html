How to use this page

Click Edit to make changes to the text of the ticker item
If you click Edit and make changes to the ticker, click Save to save these changes.
If you click Edit and want to undo the changes to the ticker, click Undo to revert any changes and go back.
Click Add to add a new ticker item below the current item
Click Remove to delete the current item
Move up will change the order (before) in which the order will display on the site
Move down will change the order (after) in which the order will display on the site
